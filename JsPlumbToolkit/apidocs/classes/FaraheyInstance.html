FaraheyInstance Class

Applies repulsive magnetism to a set of elements relative to a given point, with a specified amount of padding around the point.

Constructor

FaraheyInstance

(
  • params
)

Parameters:

  • params Object

    Constructor parameters.

    • [container] Selector | Element optional

      Element that contains the elements to magnetize. Only required if you intend to use the executeAtEvent method.

    • [getContainerPosition] Function optional

      Function that returns the position of the container (as an object of the form {left:.., top:..}) when requested. Only required if you intend to use the executeAtEvent method.

    • getPosition Function

      A function that takes an element and returns its position. It does not matter to which element this position is computed as long as you remain consistent with this method, setPosition and the origin property.

    • setPosition Function

      A function that takes an element and position, and sets it. See note about offset parent above.

    • getSize Function

      A function that takes an element and returns its size, in pixels.

    • [padding] Number optional

      Optional padding for x and y directions. Defaults to 20 pixels in each direction.

    • [constrain] Function optional

      Optional function that takes an id and a proposed amount of movement in each axis, and returns the allowed amount of movement in each axis. You can use this to constrain your elements to a grid, for instance, or a path, etc.

    • [origin] Number optional

      The origin of magnetization, in pixels. Defaults to 0,0. You can also supply this to the execute call.

    • elements Selector | String | Element

      List, or object hash, of elements on which to operate.

    • [executeNow=false] Boolean optional

      Whether or not to execute the routine immediately.

    • [filter] Function optional

      Optional function that takes an element id and returns whether or not that element can be moved.

    • [orderByDistanceFromOrigin=false] Boolean optional

      Whether or not to sort elements first by distance from origin. Can have better results but takes more time.

execute

(
  • [o] Optional origin to use. You may have set this in the constructor and do not wish to supply it, or you may be happy with the default of [0,0]
  • [options]
)

Runs the magnetize routine.

Parameters:

  • [o] Optional origin to use. You may have set this in the constructor and do not wish to supply it, or you may be happy with the default of [0,0] Number optional

    .

  • [options] Function optional

    Options to override defaults.

    • [gather] Boolean optional

      If true, will first gather the elements around the given origin before applying the magnetizer.

    • [filter] Function optional

      Optional function to indicate whether a given element may be moved or not. Returning boolean false indicates it may not.

    • [padding] Optional [x,y] Number optional

      padding values for elements.

    • [iterations] Number optional

      Optional max number of iterations to run. The greater this number, the more comprehensive the magnetisation, but the slower it runs. The default is 2.

    • [exclude] Function optional

      Optional function to return whether or not a given element should be completely excluded from the magnetisation: it neither moves, nor has any bearing on the movement of other elements.

    • [excludeFocus=false] Boolean optional

      If true, do not pad any elements around the focus point.

setConstrain

(
  • c
)

Sets the function used to constrain the movement of some element that the magnetizer wishes to relocate. The function is given an element ID and an array of [x,y] values, where each value indicates the proposed amount of movement in the given axis. The function is expected to return an array of [x,y] that indicates the allowed amount of movement in each axis.

Parameters:

  • c Function
